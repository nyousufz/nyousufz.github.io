PROGRAMMING

1A Final Project (Robot C/C++):

        For our 1A final project as part of a team of four students at University of Waterloo, we decided to build a Plotter. Our team was provided with a Lego kit to complete the project. We divided the plotter into two main parts. The first part was the tray that moved in X direction and the tool (pencil/pen) holder that moved in the Y direction. We provided the user with two different choice of tools. The plotter was able to take .SVG file from a computer and plot it on a paper. It was also able to provide the user to have, LIVE CONTROL, meaning the user could use Joystick/controller to Plot anything.
